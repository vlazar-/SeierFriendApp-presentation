SeierFriendApp

customer loyalty application




Mobile application SeierFriendApp is intended for visitors of shopping city in Graz. Visitors can become "Friend of Seiersberg" and on each visit they collects points.

There are three levels of friend:

  • Basic Friend
  • Top Friend
  • Premium Friend

Currently Evloaris have installed for terminals inside shopping city center where “Friends of Seiersberg” via NFC card can see their point status and vouchers.

The main goal of project is explore possibility to replace NFC cards with Bluetooth Low Energy technology.

Instead of NFC cards users would have all the terminal functionality on their smartphones.

After visiting Graz, Evolaris mentors defined user use case for application.

So.. the use case is:

  • User walks into a shopping city.
  • The app detects Bluetooth beacon and shows notification.
  • User opens notification and “checks in”.
  • (optional) Users browses vouchers.

Based on use case the functional requirements are:

  • User can sign in
  • User can see his points status and receive new ones
  • User can see his personal information
  • User can see available vouchers
